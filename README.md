🌦️ Ultimate Weather App

A modern and animated Weather Application built using HTML, CSS, and JavaScript.
This app allows users to search for any city worldwide and view real-time weather information with beautiful UI animations such as rain, snow, clouds, and sunshine.

The application also supports GPS-based location detection, allowing users to instantly check the weather for their current location.

🚀 Features

🔍 City Search Weather

Enter any city to get current weather data.

📍 GPS Location Weather

Automatically fetch weather using your device location.
